\chapter[光学]{\itr{Optics}{光学}}
世界因为光而五彩斑斓，万物因为光而更有生机。中学我们已经学过一些光的基础知识，例如反射、折射以及全反射，以及干涉和衍射等。在这一章节，我们将从中学的基础出发，继续深入探究光的性质及其背后的机理。并且我们将会发现更多光的美妙。

这一章节，将从几何光学以及波动光学两个部分进行光的讨论，基础要求较低，读者可以不必太过担心。
\section[几何光学]{\itr{Geometrical optics}{几何光学}}
\subsection[导论]{\itr{Introduction}{导论}}
中学我们学过一些反射和折射的性质，例如平面镜反射中入射角等于反射角，而在不同介质交界面发生的折射则满足$n=\dfrac{\sin\theta_1}{\sin\theta_2}$，其中$n$为折射率等等。

最初，物理学界对光的认知分为以牛顿为代表的“光是一种粒子”以及以惠更斯为代表的“光是一种波”。对于折射现象，牛顿的理论在光子不受力的前提下，其平行于交界面的动量应该不变，但角度的变化导致推导出了超越
光速的更快光速，这是不合常理的。而惠更斯认为光是在以太中的纵波，很明显，波峰折射后依然是波峰，波谷亦然。我们可以通过假定在两种介质中的波长不同，并以交界面处列写等式$\dfrac{\lambda_i}{\cos\theta_i}=\dfrac{\lambda_j}{\cos\theta_j}$，顺利推导出了满足麦克斯韦方程组的光速在介质中的速度较真空小的正确结论。

1850年，福柯测量到水中的光速确实小于真空光速，这一决定性的实验因此扼杀了牛顿的光粒子理论。虽然惠更斯的以太假设并没有确切的根据，但确实他发现了一个正确的结论，并且凑巧的是，这一结论与波粒二象性结论结合动量守恒假设推导的结果一致，均有：
\[n=\dfrac{c}{v}\geq 1\]

对于白光，其融合了多种不同波长的波。我们知道光速是恒定的，但光的波长以及频率则会有多种值，我们日常可见光仅是其中很小的一段范围内的光：

\end{singlefigure}

对于不同波长的光，在同一种折射中折射率是不同的，由此得到了色散现象，使得出现了彩虹、日晕等自然现象。
\subsection[反射和折射]{\itr{Reflection  and  Refraction}{反射和折射}}
反射和折射相信读者已经很熟悉了，并且对这两种现象都有一定了解。那么为什么会这样？我们需要首先介绍\itr{Fermat’s Principle}{费马原理}：

费马原理认为：光从一点到另一点是沿着光程为极值的路径传播的。即光沿光程为极小、极大或者常量的路径传播。该原理又称作极端光程定律。光程$s$是光在介质中传播的几何路程$l$与折射率$n$的乘积。表示为：
\[L = nl = ct = \int n \dif{l}\]

根据费马原理，该式应为极值，即任意一条与该路径不相等的路径都会导致整体光程变长，即：
\[\delta L = \delta \int n\dif{l} = 0\]

上式也是费马原理的数学表达式。

从费马原理出发，我们可以证明以下的反射和折射的性质，过程详见证明部分：
\begin{law}[\itr{Reflection}{反射}---\refleaftext{prove9.1}]
    光的反射定律：反射光与入射光和法线在同一平面内；入射光与反射光在法线两侧，与法线夹角相同，即入射角$\theta_1$和反射角$\theta_2$相等：
    \[\theta_1=\theta_2\]
\end{law}
\begin{law}[\itr{Refraction}{折射}---\refleaftext{prove9.2}]
    光的折射定律：折射光与入射光和法线共面；折射角与入射角的正弦之比与入射角大小无关，仅由两介质决定，当温度、压力以及光波长一致时，其比值是一个常数，等于前一介质与后一介质折射率之比，即：
    \[n_1 \sin\theta_1= n_2\sin\theta_2\]
\end{law}

说到折射率，就需要回顾一下上一章麦克斯韦方程组解出的光速了，真空中$c = \dfrac{1}{\sqrt{\varepsilon_0\mu_0}}$，在介质中的光速$v = \dfrac{1}{\sqrt{\varepsilon\mu}}=\dfrac{1}{\sqrt{\varepsilon_0\mu_0\kappa_e\kappa_m}}$，而折射率$n$定义为真空光速与介质中的光速之比，即：
\[n = \dfrac{c}{v} = \sqrt{\kappa_e\kappa_m}\]

其中一般材料的$\kappa_m \approx 1$，因此一般情况下$n\approx \sqrt{\kappa_e}$，同时易知真空折射率$n_0 = 1$。当然，某种材料的折射率并不是恒定的，而是随着波长变化的，这一性质称为光的色散（三棱镜可以分光的原因），一般情况下折射率随波长增大而减小，称为正常色散，当然反常色散的材料也存在。

另一个关于折射定律的特殊情况是全反射。
\begin{law}[\itr{Total Reflection}{全反射}]
    当光从大折射率$n'$介质入射到小折射率$n$介质的过程中，随着入射角增大，出射角将先达到90°，即光不发生折射，而是被全部反射，这一现象即全反射。临界入射角为：
    \[\theta = \arcsin(\dfrac{n'}{n})\]
\end{law}

全反射在光学中应用广泛，这一现象意味着可以将光的能量约束在一个小空间内传递，常见的光纤就是基于全反射实现的。（实际上的全反射需要用波动光学的理论进行解释，物理图像类似有限深势阱下的量子隧穿）。

我们做一个简单的总结：在几何光学视角下，由于物体尺寸远大于光的波长，因此把光作为光线看待。在该前提下，光遵循以下四条基本性质：
\begin{enumerate}
    \item 光的直线传播定律：在各向同性的均匀介质中，光在两点之间沿直线传播。
    \item 光的独立传播定律：以不同途径传播的光同时通过某点时，彼此互不影响。当各路光相遇处光强简单叠加（不考虑光的波动性的近似结果，衍射、干涉等现象不满足此定律）。
    \item 光的反射定律：反射光与入射光和法线在同一平面内；入射光与反射光在法线两侧，与法线夹角相同，即入射角与反射角相等。
    \item 光的折射定律：折射光与入射光和法线共面；折射角与入射角的正弦之比与入射角大小无关，仅由两介质决定，当温度、压力以及光波长一致时，其比值是一个常数，等于前一介质与后一介质折射率之比。
\end{enumerate}

\subsubsection[透镜与球面成像]{\itr{Lens and spherical imaging}{透镜与球面成像}}
透镜与球面成像规律算是这一部分比较重要并且相对复杂的内容。不过在普通物理学中一般只考虑满足旁轴近似\footnote{傍轴近似可以考虑为光线和光轴的夹角很小，从而可以对表达式中的三角函数等进行等价量替换，主要用于简化波场的计算}情况下的理想光学系统成像，不考虑像差。

在球面镜成像中，光线在介质的交界面处发生折射导致光偏向。先考虑单边球面透镜的情况，如下图所示：
\begin{singlefigure}[球面透镜]{chapter9_球面透镜}[0.8]
\end{singlefigure}

从Q点发出的光线经过折射后到达Q'点，我们希望得到一个尽可能简化的表达式以获得像距、物距的关系。由于推导过程较为复杂，下面直接给出结论：
\begin{law}[球面透镜的成像规律---\refleaftext{prove9.4}]
    在如上图表示的球面透镜成像中，在傍轴近似条件下，物距$u$、像距$v$和曲率半径$r$满足：
    \[\dfrac{n}{o} + \dfrac{n'}{i} = \dfrac{n'-n}{r}\]

    定义物方焦距和像方焦距为$f$、$f'$，则有：
    \[\dfrac{f}{o} + \dfrac{f'}{i} = 1\]
\end{law}

上式即常见的球面透镜成像规律，其中$\dfrac{n'-n}{r}$叫做光焦度，记为$\phi$，可表征折射球面的光学特性。

取物距$u\rightarrow \infty$，即考虑一束平行入射光，对应的像距为像方焦距：
\[v = f' = \dfrac{n' r}{n'-n}\]

同理有物方焦距：
\[u = f = \dfrac{n r}{n'-n}\]

易知：
\[\dfrac{f'}{n'} = \dfrac{f}{n} = \dfrac{1}{\phi}\]
\[f' - f = r\]

在使用这个公式时，我们需要规定一些符号，因为算出某个距离为负数是很常见的现象。

人眼观察到物体，来自于物体向周围发出的光（准确来说是物体向四周散射的光）在人眼中成像。像分为\itr{real image}{实像}和\itr{virtual image}{虚像}。这个正负号就对应着成像结果。但注意不同的教材中使用的规范不一定一致，但只要这个规范是固定的，那结果就会是正确的。
\begin{enumerate}
    \item 物（像）是实物（像），对应的物（像）距为正；物（像）是虚物（像），对应的物（像）距为负。
    \item 球心在球面与轴线交点：右侧为正，左侧为负。例如上面我们的示意图。
    \item 垂直于光轴的线段：在光轴上方为正，下方为负。
    \item 角度：沿光轴逆时针旋转为正，顺时针旋转为负。
\end{enumerate}

使用球面透镜的结论，我们定义几种放大倍率及其表达式：
\begin{singlefigure}[球面透镜成像示意图]{chapter9_三种放大倍率}[0.9]
\end{singlefigure}

(1) 横向放大倍率

横向放大倍率定义为像高与物高之比$m = \dfrac{y'}{y}$。在傍轴近似条件下结合折射定律，使用我们前面定义的规则，明显可得：
\[m = \dfrac{y'}{y} = -\dfrac{i\theta'}{o\theta} = -\dfrac{ni}{n'o} = -\dfrac{fi}{f'o}\]

(2) 轴向放大倍率

轴向放大倍率定义为像点对应物点沿轴移动量之比$\alpha = \dfrac{\dif i}{\dif o}$。对$\dfrac{f}{o} + \dfrac{f'}{i} = 1$微分，可得：
\[-\dfrac{f}{o^2} \dif o - \dfrac{f'}{i^2} \dif i = 0\]

化简得到：
\[\alpha = \dfrac{\dif i}{\dif o} = -\dfrac{i^2f}{o^2f'} = -\dfrac{i^2 n}{o ^2 n'} = -\dfrac{n'}{n}m^2\]

(3) 角放大倍率

角放大倍率定义为光轴上一个点发出的光经透镜折射后光线与光轴夹角的正切值与发出光时光线与夹角的正切值之比，可以参考球面透镜的示意图中的字母，表示为$\gamma = \dfrac{\tan u'}{\tan u}$。根据几何关系以及上面的结论：
\[\gamma = \dfrac{\tan u'}{\tan u} = \dfrac{u'}{u} = -\dfrac{o}{i} = \dfrac{n}{n'}\dfrac{1}{m}\]

从以上推导也可以得到一个关系式：
\[J = nyu = n'y'u'\]

这一关系称为拉格朗日-赫姆霍兹（Lagrange-Helmholtz）不变量。它由拉格朗日和赫姆霍兹提出,揭示了物象平面内的一个守恒量。在近轴光学中，它表示物方折射率、物高和孔径角乘积与像方对应参数乘积相等的守恒关系。该不变量源于对光学系统成像规律的数学描述，在多个透镜组合中有重要作用。

另外，以上三种放大倍率也存在关系：
\[\alpha\gamma = -m\]

这个是按照上述符号规定得到的关系式，如果换一种符号规定，对应放大倍率的符号也会相应发生变化，但绝对值或者长度大小不会发生改变。

\subsection[透镜成像]{\itr{Lens imaging}{透镜成像}}
薄透镜是对一般透镜的一种近似结果，一般透镜厚度相比其球面（曲率）半径很小，略去厚度可以简化计算，但不会引起成像结果发生实质性变化。重要公式包括磨镜者公式和牛顿公式。
\begin{singlefigure}[薄透镜成像]{chapter9_薄透镜成像}[0.9]
\end{singlefigure}

一种薄透镜如上图所示，事实上我们下面讨论的透镜无所谓其凹凸，因此当做经典的凹透镜或者凸透镜都适用。
\begin{law}[\itr{Lens maker’s equation}{磨镜者公式}---\refleaftext{prove9.6}]
    对于上述展示的薄透镜成像，设两个面的曲率半径分别为$r_1$和$r_2$。

    物方焦距：
    \[f = \dfrac{nr_1 r_2}{(n'-n_L)r_1 + (n_L - n)r_2}\]

    像方焦距：
    \[f' = \dfrac{n'r_1 r_2}{(n'-n_L)r_1 + (n_L - n)r_2}\]

    如果透镜两侧均为空气，即$n = n' = 1$，则：
    \[f = f' = \dfrac{1}{(n_L - 1)(\dfrac{1}{r_1} - \dfrac{1}{r_2})}\]
\end{law}

由磨镜者公式可知，如果$r_1 > 0$，$r_2 < 0$，即透镜中心厚度大于边缘厚度，也就是说透镜为凸透镜，则焦距为正；反之如果$r_1 < 0$，$r_2 > 0$，即透镜中心厚度小于边缘厚度，也就是说透镜为凹透镜，则焦距为负。这与我们中学学过的常识是一致的。

我们再进一步思考一种中学常见的凸透镜光路图。当时我们得到的结论是随着物体和焦点的位置关系，凸透镜会有放大、缩小以及实虚像的变化。物和像以及焦点存在焦物距和焦像距的定义。那我们定义了一个距离总得有点用，牛顿公式便阐述了透镜的焦像距、焦物距与焦距之间的关系。
\begin{singlefigure}[牛顿公式示意图]{chapter9_牛顿公式}[0.9]
\end{singlefigure}

如上图所示，焦物距定义为物点到物方焦点间的距离，如果物点在物方焦点左侧，则焦物距为正，反之焦像距为负；焦像距定义为像点到像方焦点间的距离，如果像点在像方焦点右侧，焦像距为正，反之焦像距为负。
\begin{law}[\itr{Newton’s law of refraction}{牛顿公式}---\refleaftext{prove9.7}]
    焦物距$x$、焦像距$x'$和物像方焦距$f$、$f'$满足：
    \[xx' = ff'\]
\end{law}

此外，薄透镜的横向放大倍数与球面透镜一致，代入牛顿公式可得：
\[m = -\dfrac{fi}{f'o} = -\dfrac{fx'+ff'}{f'x + f'f} = -\dfrac{f}{x} = -\dfrac{x'}{f'}\]

配眼镜的时候经常听到屈光度这个词，而眼镜又是一种凹凸透镜。事实上，屈光度定义为焦距的倒数，即：
\[P = \dfrac{1}{f}\]

\subsection[几何光学的应用]{\itr{Applications of Geometrical Optics}{几何光学的应用}}

应用一些常见的曲面透镜以及凹透镜、凸透镜等，我们可以得到非常丰富的光学仪器，例如显微镜、望远镜等。在多个连续透镜组合的过程中，在薄透镜的前提下，从几何视角我们可以结合前文提到的定义和公式得到一些结论：
\begin{singlefigure}[薄透镜成像]{chapter9_薄透镜}[0.8]
\end{singlefigure}
\begin{enumerate}
    \item 如果一个光线平行于光轴射入薄透镜，那么它的出射光线将会经过像方焦点。
    \item 如果一个光线经过薄透镜的物方焦点，那么它将从透镜中以平行于光轴的方向射出。
    \item 如果一个光线经过薄透镜的中心，那么它的出射光线方向不会改变，因为在薄透镜前提下中心附近可以视为很薄的平行界面。
\end{enumerate}
\subsubsection[人眼]{人眼}
人眼可以理想化为一个可调焦距的透镜（晶状体）以及一个像距固定的屏幕（视网膜），像距固定约为25mm。正常眼可以同通过焦距调整看清无穷远到$s_0 = 250$mm之间的物体。近视是眼睛的焦距太小，对光的汇聚能力太强，导致远点成像在视网膜前，需要使用凹透镜进行矫正；远视是眼睛的焦距太大，对光的汇聚能力太弱，导致近点成像在视网膜后方，需用凸透镜矫正。

\subsubsection[放大镜]{\itr{Magnifier}{放大镜}}
放大镜实际上就是凸透镜当物体在一倍焦距内的情况下成像，即实物呈放大正立的虚像。但放大镜的放大倍率并不是简单的横向放大倍率，而是视觉放大倍率。

人眼判断物体大小主要依赖于物体相对眼睛的角度，如下图所示：
\begin{singlefigure}[放大镜示意图]{chapter9_放大镜}[0.8]
\end{singlefigure}

放大前，人眼的明视距离为25cm，对应的角度正切值为：
\[\tan \theta = \dfrac{y}{s_0}\]

加上放大镜后，同样的物体可以移动到接近焦点的位置，物和虚像在人眼观察到的同一个角度上，其正切值为：
\[\tan \theta' = \dfrac{y}{f}\]

因此放大镜的倍率为：
\[m_{\theta} = \dfrac{\tan \theta'}{\tan \theta} = \dfrac{s_0}{f}\]

\subsubsection[显微镜]{\itr{Microscope}{显微镜}}
显微镜由两个凸透镜组成，即物镜和目镜。其放大倍率为两个镜片的放大倍率之积。示意图如下：
\begin{singlefigure}[显微镜示意图]{chapter9_显微镜}[0.8]
\end{singlefigure}

显微镜中，物镜和目镜焦点存在一个间距，即上图所示的$\Delta$。当物体处于物镜的物方焦点外侧时，经物镜的倒立放大的实像落于目镜的焦点内侧，再经目镜的虚像落于明视距离之外。此过程中的放大倍数与放大镜一致，采用视角放大率定义。在明视距离处的张角和之前一致，为：
\[\tan\theta = \dfrac{y}{s_0}\]

设最终的虚像的大小为$y'$。明视距离作为一处临界条件，对应于最终成像光线接近平行。考虑到虚像在光轴下侧，从光轴到光线的角度旋转为逆时针，定义为负数（顺时针为正数）\footnote{注意这里的正负号与之前的约定刚好相反，这是因为视角是逆光线观察的}。则显微镜处理后的视角为：
\[-\tan \theta' = \dfrac{-y'}{f_e}\quad\Rightarrow\quad M = -\dfrac{y'}{y}\dfrac{s_0}{f_e} = V_0 m_\theta\]

$V_0$为物镜的横向放大倍数，$M_E$为目镜的视角放大率。物镜的像在光轴下方，长度为$-y'$。从几何关系或者前面得到的薄透镜横向放大倍数可以得到：
\[V_0 = -\dfrac{x'}{f'} = -\dfrac{\Delta}{f_o}\]

得到：
\[M = -\dfrac{\Delta}{f_o}\dfrac{s_0}{f_e}\]

这里的放大倍数为负数，表示为像是倒立的。
\subsubsection[望远镜]{\itr{Telescope}{望远镜}}
望远镜用于观察很远处的物体，因此入射光线可以近似视为平行光\footnote{但为了便于表示，我们会给部分光线一个夹角以便于在有限的绘图区域内最终成像}。望远镜种类有很多，其中最主要的开普勒望远镜和伽利略望远镜两种。分别记物镜焦距为$f_o$，目镜焦距为$f_e$。

开普勒望远镜的物镜和目镜均为凸透镜，一束平行光入射后仍将以平行光射出目镜。如下图所示：
\begin{singlefigure}[开普勒望远镜示意图]{chapter9_开普勒望远镜}[0.9]
\end{singlefigure}

根据开普勒望远镜的要求，平行光入射会汇聚到物镜的焦点，而为了以平行光出射必须经过目镜的焦点，因此物镜和目镜的焦点是重合的，满足：
\[f_o + f_e = L\]

另外从上图可以定性发现，开普勒望远镜成像为倒立的虚像。我们进一步讨论其放大倍数。同理，这也表示为视角放大率。则有：
\[\tan \theta = \dfrac{-y}{s_0} = \dfrac{y}{f_o}\qquad  -\tan \theta' = \dfrac{-y'}{f_e}\]

因此：
\[M = -\dfrac{f_o}{f_e}\]

伽利略望远镜的物镜为凸透镜，但其目镜是一个凹透镜。同样地，它也要求平行光入射物镜必然平行光射出目镜。因此，其焦距满足关系：
\[f_o - f_e = L\]

其余推导与开普勒望远镜类似，读者可以通过视角放大率继续探究伽利略望远镜的放大倍数。

\section[干涉]{\itr{Interference}{干涉}}
干涉是光学中的重要现象，很多光学现象都可以用干涉解释，同时很多光学器件以及光学相关的应用也是以干涉为原理的，例如电光调制（马赫赠的干涉仪）、引力波探测（干涉测距）等等。几何光学部分提到光的独立传播定律在这时不成立，这种情况下物体的大小与光波长相近，光的波动性便不能忽略。我们需要从波的角度解释光的现象，也就有了干涉、衍射等现象。

干涉在本质上就是波在以振幅进行线性叠加后导致其和，也即强度（振幅的平方）发生与线性叠加不同的形式，其在任何波的叠加中都存在，包括上学期的机械波、这里的光波以及量子力学中的概率波。不过光在干涉中有一个特殊点，就是光的频率极高，高到目前还不存在可以响应光频率的探测器，因此在讨论干涉条纹或者光强度时，这里说的都是时间平均后的结果，这也是干涉形成的条纹不随时间变化的原因。

从上一章的推导可知光是电磁波，是矢量波。设$P$点相位为$\varphi(P)$，则：
\[E = E_0(P) \cos(\omega t - \varphi(P))\]
\[H = H_0(P) \cos(\omega t - \varphi(P))\]

由于电场与磁场间存在固定的关系，在后面的计算中为了方便，仅用电场表示光波。为了分离表达式中的时间，简化计算，这里引入波的复数表示。对于$E = E_0(P) \cos(\omega t - \varphi(P))$，其表达式等价于$\tilde{E} = E_0(P) e^{\pm i(\omega t - \varphi(P))}$的实部。这里正负号的选择只要统一即可，我们暂时取负号。我们还可以定义复振幅$\tilde{E}_0(P) = E_0(P) e^{i\varphi(P)}$，因此波和光强可表示为：
\[\tilde{E} = \tilde{E}_0(P) e^{-i\omega t}\qquad I = |E_0(P)|^2 = \tilde{E}_0(P) \tilde{E}_0(P)^*\]

这里的光强是一个相对值，在介质统一的情况下可以用此表示光强的相对大小，并不是绝对值。

对于平面波，复振幅可表示为：
\[\tilde{E}_0(P) = E_0(P) e^{i(k_x x + k_y y + k_z z)}\]

我们后续的推导将以波的复数表示为基础。
\subsection[干涉条件]{\itr{Interference Requirement}{干涉条件}}
空间中两束波在$P$点的振动分别表示为：
\[\tilde{U}_1(P,t) = A_1 e^{i\varphi_1(P)} e^{-i\omega t}\qquad \tilde{U}_2(P,t) = A_2 e^{i\varphi_2(P)} e^{-i\omega t}\]

线性叠加可得：
\[\tilde{U}(P,t) = (A_1 e^{i\varphi_1(P)} + A_2 e^{i\varphi_2(P)}) e^{-i\omega t}\]

计算强度：
\begin{align*}
    I(P) &= \tilde{U}(P,t)\cdot \tilde{U}(P,t)^* = (A_1 e^{i\varphi_1(P)} + A_2 e^{i\varphi_2(P)})\cdot (A_1 e^{-i\varphi_1(P)} + A_2 e^{-i\varphi_2(P)})\\
    &= A_1^2 + A_2^2 + A_1A_2 (e^{i(\varphi_1(P) - \varphi_2(P))} + e^{-i(\varphi_1(P) - \varphi_2(P))})\\
    &= A_1^2 + A_2^2 + 2A_1A_2 \cos(\varphi_1(P) - \varphi_2(P))\\
    &= I_1(P) + I_2(P) + 2\sqrt{I_1(P)I_2(P)} \cos(\varphi_1(P) - \varphi_2(P))
\end{align*}

可以看到，除了两束波的强度的加和之外，还有一项与$\cos(\varphi_1(P) - \varphi_2(P))$有关，这一项即为干涉项。只要分析清楚不同情况下相位差的变化，就可以分析清楚干涉现象。后面的杨氏干涉、等倾干涉、等厚干涉等不同的干涉现象就是不同相位差下的强度变化的具体表示。
\begin{law}[\itr{Interference Requirement}{干涉条件}]
    波的干涉是由于波的叠加引起强度重新分布的现象。为了形成稳定的干涉，需要：
    \begin{enumerate}
        \item 两束波必须频率相同，以确保叠加后的光的强度不随时间变化，否则经时间平均后干涉项强度为0。
        \item 两光波的电场矢量偏振态相同（更准确应该是存在同向分量），以确保两光波的复振幅可以线性叠加。
        \item 两束波有确定的相位差。目的与同频率的要求一致。
    \end{enumerate}
\end{law}

当然，如果两光波频率相差不大，干涉后会出现拍频的现象，这也是测量频率差的一种重要手段。还是利用前面的两束波，假设其在一维平面运动，即：
\[\tilde{U}_1(P,t) = A_1 e^{ik_1 x} e^{-i\omega_1 t}\qquad \tilde{U}_2(P,t) = A_2 e^{ik_2 x} e^{-i\omega_2 t}\]
\[I(P,t) = I_1 + I_2 + 2\sqrt{I_1I_2}\cos[(k_1 - k_2)x - (\omega_1 - \omega_2)t]\]

明显，在确定的任意一个$x$处，将会以一个很小的频率振动。这也被称为拍频。上述现象可以如下图表示：
\begin{singlefigure}[拍频]{chapter9_干涉拍频}[0.95]
\end{singlefigure}

虽然我们用平面波的形式表示光波，但这是理想情况，要求光可以延伸到无穷远，且持续时间无穷长，显然实际情况不是这样，光的持续时间有限，因此光的波列长度有限，只有在相干长度内才会发生干涉现象。

    满足干涉条件时，在$L\gg d$的近似条件下，明暗条纹的分布与其距离中心点的距离的关系为：

        \dfrac{L}{d}m\lambda \quad \text{m为整数} & \text{明纹}\\[1em]
        \dfrac{L}{d}(m+\dfrac{1}{2})\lambda \quad \text{m为整数} \quad & \text{暗纹}
    \end{cases}\]
\end{law}

\subsection[薄膜干涉]{\itr{Thin film interference}{薄膜干涉}}
薄膜干涉示意图如图所示，其利用薄膜上层与下层的反射光进行干涉，是典型的分振幅干涉。薄膜干涉在生活中很常见，像肥皂泡上面的彩色条纹、油膜都是薄膜干涉。

在薄膜干涉中，两束光的相位差由反射过程提供，但在反射界面上会有相位突变的情况。这是在分析薄膜干涉的时候必须留意的一点，半波损失的存在与否将直接导出完全相反的结论。
\begin{singlefigure}[薄膜干涉]{chapter9_薄膜干涉}[0.6]
\end{singlefigure}

\begin{law}[\itr{Half-Wave Loss of Reflecting Light}{反射光的半波损失}---\refleaftext{prove9.9}]
    在不同介质的界面处反射时，如果$n_1 < n_2$，那么那么反射光相对于入射光将会有$\pi$的相位差（又称半波损失）；反之，如果$n_1 > n_2$，那么反射光相对于入射光相位相同。

    折射光线不会出现半波损失
\end{law}

在物理图像上，这一过程可以类比机械波中不同边界下的反射，$n_1<n_2$可以类比固定边界机械波反射，$n_1>n_2$可以类比自由边界机械波反射。

薄膜干涉有两种经典形式，分别为等倾干涉和等厚干涉。等倾干涉如下图所示：
\begin{singlefigure}[等倾干涉及其条纹]{chapter9_等倾干涉及其条纹}[0.9]
\end{singlefigure}

在具体推导相位差表达式之前，需要先说明三个问题。

首先是等倾干涉中“等倾”二字的含义：“倾”表示入射光的倾斜程度，即入射角，因此“等倾”表示的就是相同的入射角，换句话说就是同意入射角下的光在干涉条纹中产生一个相位差，干涉条纹是由入射角不同引起的。

第二个问题是条纹的观测：从图中（或者简单的反射定律计算）可知，两次反射的光互相平行，在一般情况下无法观测到，只有当观测点在无穷远或者有透镜聚焦的情况下才可以观测到干涉条纹，这也是示意图中透镜存在的原因。

第三个问题是多次反射的情形：既然上下两个面的第一次反射会干涉，那么为什么不考虑下表面第二次、第三次反射后的光与前面几次反射的光的干涉呢？这主要是因为一般情况下（第一次反射率较小），图中画出的两次反射光强度较大且接近，而其余的多次反射光强均较小，因此可以忽略不计。以空气（$n=1$）和玻璃（$n\approx1.5$）界面为例，上表面反射率为4\%，上表面透过下表面反射的反射率$R=(1-4\%)*4\%=3.84\%$二者几乎相等，而下表面第二次反射的反射率以相同的方法计算可知不到0.01\%，远小于前两次。当然如果全部考虑，计算结果会更加准确，在某些情况下为了计算准确会考虑多次反射后的多光束干涉，但在这里考虑两次反射光的干涉就足够了。

等倾干涉时干涉光的强度与杨氏双缝干涉的表达式一致，明暗纹条件为：
\begin{law}[\itr{equal inclination interference}{等倾干涉}]
    不考虑半波损失时，明暗纹分布为：
    \[2nh\cos i = \begin{cases}
        m\lambda \quad \text{m为整数} & \text{明纹}\\
        (m+\dfrac{1}{2})\lambda \quad \text{m为整数} \quad & \text{暗纹}
    \end{cases}\]
\end{law}

我们上面的结论忽略了半波损失，因此建议读者自行根据光程和波的基础知识推导，尤其在考试时务必注意不要忽略半波损失的可能性。

从上式可以看出，中心位置（$i=0$）的条纹级次最高，越往外的条纹级次越低，因此当h增大或者折射率n增大时，条纹将从中心冒出；反之，条纹将从中心收缩，该现象在迈克尔逊干涉仪中可以更直观的展现。根据透镜成像规律，对应条纹半径$r = f\tan i$，因此相邻两条纹半径为：
\[r_{m+1} - r_m = f(\tan i_{1_{m+1}} - \tan i_{1_m})\approx f(i_{1_{m+1}}-i_{1_m}) = \dfrac{fn}{n_1}(i_{1_{m+1}}-i_{1_m})\]

而根据我们在干涉部分的结论，我们对公式取泰勒展开到一阶项，可得：
\[\cos i_{m+1} - \cos i_m = \dfrac{\lambda}{2nh} = -\sin i_m(i_{m+1}-i_m)\]

因此：
\[r_{m+1} - r_m = -\dfrac{f\lambda}{2n_1 h\sin i_m}\]

从中可以看出，角度越大，相邻两个条纹的间隔越小，也即半径越大，条纹越密集。

等倾干涉的一个实际应用的例子是迈克尔逊干涉仪，迈克尔逊干涉仪在历史上有众多应用，包括证明以太不存在的迈克尔逊-莫雷实验，以及引力波探测实验等等。其结构如下图所示：
\begin{singlefigure}[迈克尔逊干涉仪]{chapter9_迈克尔逊干涉仪}[0.8]
\end{singlefigure}

由两个垂直的反射镜$M_1$、$M_2$与一个分光镜$G_1$构成，其中$G_2$为补偿板，补偿由$G_1$在另一条路径中引起的光程差，如果两臂间距离为$d$，那么光程差表示为：
\[\Delta L = 2nd\]
\subsection[等厚干涉]{\itr{Equal Thickness Interference}{等厚干涉}}
等厚干涉是另一种分振幅的干涉，其是楔形板上下表面反射光的干涉，如下图所示。这里“等厚”的含义是厚度相同的地方干涉的相位相同。注意到这种干涉形式的观测不需要透镜，因为两束反射光之间在空间中某一点（一般在上表面附近）相交，可以形成条纹。
\begin{singlefigure}[等厚干涉]{chapter9_等厚干涉}[0.8]   
\end{singlefigure}

与等倾干涉类似，只要计算得到相位差（光程差）就可以描述等厚干涉的条纹分布情况。由于一般情况下等厚干涉的楔形平板夹角很小，可以近似利用上面提到的等倾干涉（平行平板）的相位差（光程差）近似，即:
\[\varphi_1(P) - \varphi_2(P) = \dfrac{2\pi}{\lambda}2nh\cos i\]

这里依旧没有考虑半波损失。在实际应用中，等厚干涉情况下入射光一般垂直楔形平板入射，因此常用的相位差表达式考虑角度$i$近似为0。
\begin{law}[\itr{Equal Thickness Interference}{等厚干涉}]
    不考虑半波损失，小倾角且垂直入射时，明暗纹分布为：
    \[2nh = \begin{cases}
        m\lambda \quad \text{m为整数} & \text{明纹}\\
        (m+\dfrac{1}{2})\lambda \quad \text{m为整数} \quad & \text{暗纹}
    \end{cases}\]
\end{law}

从上面的式子可以看出，高度越高，条纹的级次越高，针对上图即越靠右，条纹的级次越高。由于等厚干涉的条纹分布在楔形平板表面，因此平板夹角与高度之间存在下面的关系：
\[h = x\theta\]

其中$x$是平板横向的距离，$\theta$是楔形平板夹角，由于$\theta$很小，可做上面的近似。将上式代入干涉条纹公式可得对应的横向的条纹周期，并可以已知条纹周期求出楔形平板的夹角：
\[\Delta x\theta = \Delta h = \dfrac{\lambda}{2n}\quad\Rightarrow\quad \Delta x = \dfrac{\lambda}{2n\theta}\]

\section[衍射]{\itr{Diffraction}{衍射}}
光波在传播过程中遇到障碍物时，会偏离原来的传播方向入射到障碍物的几何阴影区内，并在几何阴影区与几何照明区内形成光强的不均匀分布，这种现象称为光的衍射。与干涉现象相同，其它类型的波也存在衍射现象。衍射现象在生活中非常常见，大到限制光刻机光刻尺寸的衍射极限；小到对着光源拍照时产生的星芒，都是衍射现象的体现。下面将从单缝衍射、多缝衍射、光栅色散等方面介绍衍射现象。注意，本章中所讲到的衍射大多只针对光屏距离缝隙非常远或者在缝后有透镜聚焦的情况，即夫琅禾费衍射（生活中最常见），对于光屏距离缝隙较近情况下的菲涅尔衍射则几乎不涉及。下图从上到下分别是近孔区域的光学传播、夫琅禾费衍射、菲涅尔衍射的示意图。
\begin{singlefigure}[衍射]{chapter9_衍射}[0.75]
\end{singlefigure}

衍射在本质上与干涉一致，都是光以波动的形式叠加之后的结果，衍射可以利用麦克斯韦方程组以及惠更斯原理计算得到，或者利用基尔霍夫衍射理论严格推导得出，但基于图像的直观相位分析会更加形象，更加易于理解，因此后面的推导基本上是基于对图像的直观分析的。

在分析之前要引入复数运算中的一个重要的概念，相幅矢量。相幅矢量是复数的一种形象表示，可以利用矢量快速进行复数的加减法，并与三角函数的加减法相对应。具体内容读者可参考三角函数、复数与矢量关系。

\subsection[单缝夫琅禾费衍射]{\itr{Single-Slit Fraunhofer Diffraction}{单缝夫琅禾费衍射}}
单缝衍射又称夫琅禾费衍射。单缝衍射的示意图如下图所示，在计算过程中可以将其视为无穷个光束的干涉（本质都是光以波动的形式叠加）。
\begin{singlefigure}[单缝衍射]{chapter9_单缝衍射}[0.8]
\end{singlefigure}

从上一节可知，干涉的核心是找到两束光之间相位差的关系，衍射也是一样，这里对于光的描述就简化为其振幅和相位（为了简化计算，这里认为缝顶端相位为零，如果选取一个特定的相位值不会影响最终光强的计算结果）。入射光振幅相同，相位由光程差决定。但是我们需要计算波的求和结果，这么一整个孔面上每一处地方都有波，我们该如何进行求和操作？我们首先介绍一下半波带法。

前面我们提到过，惠更斯等人将光视为一种波，波的前进是前一组波激发下一时刻的位置上的波动，整个光按照波面前进。菲涅尔吸收了惠更斯的每一个波扰动周围形成次波的观点，提出从一点处产生的波形成的次波面作为新的波源，空间中一点处的振动即为这个波面上每一个波面微元产生的波的叠加，这被称为惠更斯-菲涅尔原理。当然，菲涅尔的理论是存在不足的，例如他凭直觉认为能量随着波面只能前进，不存在后退的次波，这后续被基尔霍夫用数学理论加以完善。

但是这一原理需要计算积分过程，我们将其划分为有限个微元，也就是按照半波相位差把一个面上划分为多个不同的波，这样积分过程便变成了求和过程。正如上图所示，我们把一个缝划分为$N$条条纹，相邻之间相差半波长，最终的叠加结果即视为这些波的叠加。因此，从上到下每一个小缝对应的场强的复数表示如下：
\begin{align*}
    E_1 &= E_0 e^{i\cdot 0}\\
    E_2 &= E_0 e^{i\cdot \frac{2\pi}{\lambda}\frac{a}{N}\sin \theta}\\
    E_3 &= E_0 e^{i\cdot 2\cdot \frac{2\pi}{\lambda}\frac{a}{N}\sin \theta}\\
    &\cdots\\
    E_N &= E_0 e^{i\cdot (N-1)\cdot \frac{2\pi}{\lambda}\frac{a}{N}\sin \theta}
\end{align*}

这里$\dfrac{2\pi}{\lambda}\dfrac{a}{N}\sin \theta$为相邻两缝的相位差，其中$\dfrac{a}{N}$是细分的小缝间距，近似于无穷小。接下来我们用相幅矢量的方法叠加，取相位差$\Delta \varphi = \dfrac{2\pi}{\lambda}\dfrac{a}{N}\sin \theta$：
\begin{singlefigure}[衍射的相幅矢量]{chapter9_衍射的相幅矢量}[0.6]
\end{singlefigure}

从图中可以看到，最终这个无穷级数的和为$A_0$矢量代表的复数，其幅角：
\[\alpha = \dfrac{1}{2}\dfrac{2\pi}{\lambda}a\sin \theta = \dfrac{\pi}{\lambda}a\sin \theta\]

其幅值表示为线段MN的长度，为：
\[\overline{MN} = 2R\sin\alpha = 2\dfrac{\overset{\LARGE{\frown}}{MN}}{2\alpha}\sin \alpha = NE_0\dfrac{\sin\alpha}{\alpha} = E_m\dfrac{\sin\alpha}{\alpha}\]

\begin{law}[单缝夫琅禾费衍射]
    单缝夫琅禾费衍射中，复振幅表示的光场强为：
    \[E_{\theta} = E_0\dfrac{\sin\alpha}{\alpha}e^{i\cdot\frac{\pi}{\lambda}a\sin \theta}\]

    其中$E_0$为$\alpha = 0$时的和场强，对应光强满足：
    \[I_{\theta} = I_0\left(\dfrac{\sin\alpha}{\alpha}\right)^2\qquad \alpha = \dfrac{\pi}{\lambda}a\sin \theta\]

    $\alpha = 0$时取到最大光强，当$\alpha = \dfrac{\pi}{\lambda}a\sin \theta = m\pi$时为暗纹。
\end{law}

暗纹也可以从相幅矢量图中观察得到，此时相幅矢量围成一个圆，因此其和为0。单缝夫琅禾费衍射的电场强度和光强随$\alpha$的变化如下图所示。其中主极大半角宽度$\Delta \theta \approx\sin\theta = \dfrac{\lambda}{a}$，光屏上的弥散斑的宽度$\Delta y = f\Delta \theta=f\dfrac{\lambda}{a}$。因此$a$越大，$\lambda$越小，$\Delta \theta$越小，对应的光屏上的$\Delta y$越小，反之越大。
\begin{singlefigure}[单缝夫琅禾费衍射的电场强度和光强随$\alpha$的变化]{chapter9_单缝衍射的电场强度和光强随alpha的变化}[0.9]
\end{singlefigure}

\subsection[夫琅禾费圆孔衍射]{\itr{Fraunhofer circular aperture diffraction}{夫琅禾费圆孔衍射}}

\[I_{\theta} = \left(\dfrac{2J_1 (k a \sin \theta)}{ka\sin \theta}\right)^2\]

其中$J_1$为一阶贝塞尔函数，$k = \dfrac{2\pi}{\lambda}$，$a$是圆孔半径，$\theta$是倾斜角度，与单缝衍射的定义一致。夫琅禾费圆孔衍射如下图示意：

\section[多缝衍射]{\itr{Multi-slit diffraction}{多缝衍射}}
\subsection[双缝干涉与衍射的结合]{\itr{Double-slit Interference and Diffraction Combined}{双缝干涉与衍射的结合}}
多缝衍射即在考虑缝宽度的情况下多个缝之间光波的叠加的结果，示意图如下图所示。相邻两缝之间间距为$d$，缝宽为$a$。在计算中可以将多缝衍射的结果视为单个缝衍射得到的电场再进行叠加。
\begin{singlefigure}[多缝衍射]{chapter9_多缝衍射}[0.99]
\end{singlefigure}
\begin{law}[\itr{Multi-slit diffraction}{多缝衍射}---\refleaftext{prove9.11}]
    多缝衍射中，复振幅表示的光场强为：
    \[E = E_0\dfrac{\sin\alpha}{\alpha}e^{i\alpha}\dfrac{\sin(N\beta)}{\sin\beta}e^{i(N-1)\beta}\]

    其中$\alpha = \dfrac{\pi}{\lambda}a\sin \theta$，$N$为缝的个数，$\beta = \dfrac{\pi}{\lambda}d\sin \theta$。

    对应的干涉光强为：
    \[I = I_0\left(\dfrac{\sin\alpha}{\alpha}\right)^2\left(\dfrac{\sin(N\beta)}{\sin\beta}\right)^2\]
\end{law}
上面的公式中，$\dfrac{\sin\alpha}{\alpha}$ 和 $\dfrac{\sin(N\beta)}{\sin\beta}$ 分别表示单缝衍射和多缝干涉的贡献。

先考虑多缝干涉项，易知当$\beta = \dfrac{\pi}{\lambda}d\sin \theta = m\pi$时，多缝干涉项贡献的光强度取到最大值，称为主极大值，此时有：
\[d\sin\theta = m\lambda\qquad \left(\dfrac{\sin(N\beta)}{\beta}\right)^2 = N^2\]

当$N\beta$为$\pi$的整数倍，但 $\beta$不为$\pi$的整数倍时，干涉项贡献的光强度为0，取到极小值。即：
\[\beta = (m+\dfrac{m'}{N})\pi\qquad d\sin\theta = (m + \dfrac{m'}{N})\lambda\]

其中$m'$为$1 \sim N-1$之间的整数。我们也可以进一步很容易地看出，相邻两个主极大之间存在$N-1$个零值，相邻两个零值之间存在$N-2$个次极大值。仅针对干涉项，当$N$取到2时，退化为一般的双缝干涉。干涉部分的图像如下所示：
\begin{singlefigure}[多光束干涉，N=4]{chapter9_多光束干涉}[0.99]
\end{singlefigure}

由于多缝衍射光强表达式是单缝衍射项与多光束干涉项相乘，因此考虑衍射效应的干涉图样如下，是衍射与多光束干涉图像的叠加。下图展示了这种叠加的效果：
\begin{singlefigure}[考虑衍射效应的光束干涉，d=3a]{chapter9_考虑衍射效应的光束干涉}[0.99]
\end{singlefigure}
我们可以发现，$m=3$时的干涉主极大与衍射的一级零点相重合，根据衍射零点表达式$a\sin \theta = m\lambda$可知，$d = 3a$。

另外，根据主极大和多缝衍射零点的表达式，我们可以得到多缝衍射主极大的半角宽度，即主极大的角度与相邻零点间的角度差$\Delta \theta$，可由下式计算：
\[d\sin \theta_1 - d\sin \theta_2 = (m+\dfrac{1}{N})\lambda - m\lambda\quad\Rightarrow\quad d\cos\theta \cdot\Delta\theta = \dfrac{\lambda}{N}\]

因此：
\[\Delta\theta  = \dfrac{\lambda}{Nd\cos\theta}\]

可以看到波长越短，N越大，主极大宽度就越小，恰如下图所示：

\subsection[衍射光栅]{\itr{Diffraction grating}{衍射光栅}}
从多缝衍射的主极大表达式$d\sin\theta = m\lambda$可以看出，角度$\theta$和波长$\lambda$相关，因此，基于多缝衍射的原理可以实现对不同波长的光进行分离的功能。通常把由大量等宽等间距的狭缝构成的光学元件叫做光栅，其最重要的应用就是用作分光原件，分离不同波长的光。从光栅的结构也容易看出其是基于多缝衍射设计得到的，下面将据此以透射光栅（即多缝衍射）为例介绍光栅相关的特性。

多缝夫琅禾费衍射衍射图中主极大的表达式$d\sin\theta = m\lambda$在光栅中称作光栅方程，其中$d$为相邻缝之间的间距，$\theta$为出射角度，$\lambda$为波长。如果入射光是以角度$\varphi$入射，如下图：
\begin{singlefigure}[光栅光程差]{chapter9_光栅光程差}[0.6]
\end{singlefigure}

那么根据两光束之间的光程差可得光栅方程为：
\[d\sin \varphi - d\sin \theta = m\lambda\]

这里角度都是角度的绝对值，如果规定正负方向，则表达式正负可能不同，但最终计算结果相同。这里重点是画图分析相邻两缝之间的光程差。

光栅最重要的用途是分光，该结构的分光能力（色散能力）定义为单位波长差下两光分离的角度差，即$\dfrac{\Delta \theta}{\Delta \lambda}$，由光栅方程两边微分可得：
\[d\cos \theta\Delta \theta  = m\Delta \lambda\quad\Rightarrow\quad \dfrac{\Delta \theta}{\Delta \lambda} = \dfrac{m}{d\cos\theta}\]

但在关心光栅色散本领的同时，也要关注色散之后两波长的光斑能否被探测器（只响应光强）分辨出来。这一指标由参数色分辨本领A所表征，定义色分辨本领：
\[A\mgnote{部分课件中也写作$R$等其他字母} = \dfrac{\lambda}{\Delta \lambda}\]

其中$\lambda$是中心波长，$\Delta \lambda$是光栅所能分辨的最小波长。根据瑞利判据，当一个光斑的极大值与另一个光斑的零点重合时，二者恰好可以被分辨。
\begin{law}[\itr{Resolving power of diffraction grating}{光栅的分辨能力}---\refleaftext{prove9.12}]
    对于一个光栅线数$N$，第$m$级衍射下的色分辨能力为：
    \[A = \dfrac{\lambda}{\Delta \lambda} = mN\]
\end{law}

上式可见，光栅的分辨能力与光栅的参数$d$，即栅格之间的距离无关，而一般情况下$m$会取到1、2这种比较下的级次，否则衍射光强会很弱，因此$N$越大的光栅色分辨能力越好。但在实际上，由于光栅总尺寸是固定的，因此$d$与$N$并非完全独立。

衍射效应在很多方面都有应用，包括利用X射线衍射，即将晶体作为光栅，利用X射线等通过晶体后的衍射图样判别晶体的结构。其中对于相邻平行光根据入射角度不同，有不同的光程差。下图所示的情况光程差为$2d\sin\theta = m\lambda$。

\end{singlefigure}

\section[偏振]{\itr{Polarization}{偏振}}
前面几节的内容主要考虑光作为波的特性，但没有考虑其作为矢量波的特点。解麦克斯韦方程组可以知道，光的电场振荡方向只要垂直于传播方向即可，因此其可以存在不同的取向，即不同的偏振态。偏振在生活中、科研中也存在非常多应用，包括3D电影眼镜、测量材料参数的椭偏仪等等。本节将简要介绍偏振相关的内容，包括不同的偏振态、反射过程中的偏振以及双折射等内容。
\subsection[线偏振]{\itr{Linear polarization}{线偏振}}
线偏振光即光的电场矢量方向不随光传播而改变的光。线偏振光可以由自然光经过一个偏振选择的器件，如\itr{Polarizing sheets}{偏振片}或\itr{Polarizer}{偏光片}产生，它可以吸收与其方向垂直的光对应的电场。自然光经过偏振片后其强度变为原来的一半。
\begin{singlefigure}[线偏振光和偏振片]{chapter9_线偏振光和偏振片}[0.8]
\end{singlefigure}

当光强为$I_1$的线偏振光透过与其偏振方向夹角$\theta$的偏振片时，将入射光沿垂直偏振片透过方向、平行偏振光透过方向分解，易推导出出射光的光强$I_2$的表达式。
\begin{singlefigure}[经过多个偏振片下的偏振光]{chapter9_多个偏振片下的偏振光}[0.8]
\end{singlefigure}

\begin{law}[\itr{Law of Malus}{马吕斯定律}]
    对于上图的光经过存在不同方向的两个偏振片，光强满足：
    \[I_2 = I_1\cos^2\theta\qquad E_2 = E_1\cos\theta\]
\end{law}

除了线偏振态、自然光之外，光还有很多其他偏振态。当$x$方向偏振与$y$方向偏振场强相等，相位相差$\pm\dfrac{\pi}{2}$时的光即为圆偏振光；当$x$方向偏振与$y$方向偏振场强相等，相位相差某个值但不是$\pm\dfrac{\pi}{2}$或者$\pm\pi$时的光即为椭圆偏振光，如下图所示:
\begin{singlefigure}[不同类型的偏振光]{chapter9_不同类型的偏振光}[0.99]
\end{singlefigure}

对于任意的完全偏振光，与其他矢量一样，我们都可以将其表示为x轴分量和y轴分量两个线偏振光的叠加，即$E = E_x \hat{x}_0 + E_y \hat{y}_0 = a_1 e^{i(\alpha_1 - \omega t)}\hat{x}_0 + a_2 e^{i(\alpha_2 - \omega t)}\hat{y}_0$，由于决定偏振态的核心是分量的振幅和相位，即：
\[\tilde{E}_x = a_1 e^{i\alpha_1}\qquad \tilde{E}_y = a_2 e^{i\alpha_2}\]

因此我们定义琼斯矢量，它可以描述任意的完全偏振光：
\[\left(\begin{matrix}
    \tilde{E}_x\\
    \tilde{E}_y
\end{matrix}\right) = \left(\begin{matrix}
    a_1 e^{i\alpha_1}\\
    a_2 e^{i\alpha_2}
\end{matrix}
\right)\]

除了完全偏振光之外，还有自然光、部分偏振光这两类，对于这两类光的描述，通常采用偏振度$P$，即：
\[P = \dfrac{I_{max} - I_{min}}{I_{max} + I_{min}}\]

从中可知完全偏振光$P=1$，自然光$P=0$。其实有一种表示方法——斯托克斯矢量可以用来描述任意偏振态的光，包括完全偏振光、部分偏振光以及自然光，感兴趣的同学可以了解。

自然光是生活中十分常见的光的类型，像一般的太阳、蜡烛、灯泡等发的光都是自然光，他们的偏振态完全随机，也即每个方向的偏振都会存在，完全随机分布。因此自然光没有确定的电场方向，不同偏振下的相位差也不确定。当自然光透过任意方向的偏振片后，其强度都为原来光强的一半（不考虑偏振片吸收）。

\subsection[反射偏振]{\itr{Polarization by reflection}{反射偏振}}
前面对反射的分析只针对强度，这里将进一步分析不同偏振态的反射响应。对于玻璃或者其他介质材料中，存在一个入射角，当自然光入射时，得到的反射光仅有电场垂直于入射面（入射光与法线形成的平面）的偏振态（$s$波）可以被反射，而电场在入射面内的偏振态（$p$波）则被完全吸收，如下图所示，这个角度称作布鲁斯特角，此时透射光与反射光互相垂直。
\begin{singlefigure}[反射过程中的偏振]{chapter9_反射偏振}[0.8]
\end{singlefigure}
\begin{law}[\itr{Brewster’s angle}{布儒斯特角}]
    入射角$\theta_p$为布儒斯特角时，折射角$\theta_r$，反射光和折射光互相垂直，反射光完全偏振，折射光部分偏振，此时
    \[\theta_p + \theta_r = \pi/2\qquad n_1 \sin\theta_p = n_2 \sin\theta_r\]
    可得：
    \[\tan \theta_p = \dfrac{n_2}{n_1}\]
\end{law}

布鲁斯特角可以用于产生偏振光，当自然光经过多次布鲁斯特角的反射之后，透射光将以$p$偏振为主导，$s$偏振大部分被反射损耗，这正是上面的示意图所表示的含义。

\subsection[双折射]{\itr{Double Refraction}{双折射}}
当我们把一块冰洲石（一种方解石）放在写有字的纸上，我们会发现双重影像。对于一般的厚玻璃我们一般会发现一个比纸面稍高的像，这是由于折射引起的，那么在方解石中发现两个像的现象表明光在其中存在不同的折射程度，这种现象被称为双折射。

前面我们一直默认介电常数与方向无关，是一个常数，这在针对各向同性材料时是成立的，但对于一些晶体这一假设往往不成立。直观来说，晶体在不同方向上的介电常数不同，因此对应不同偏振态的电场的折射率也不同，对应的光线轨迹也不相同。一些材料中光的折射率和光速取决于介质中光的传播方向以及偏振态，这被称为光学各向异性。在这一现象中，我们主要讨论两种光：\itr{Ordinary ray (o-ray)}{寻常光（o光）}和\itr{Extraordinary ray (e-ray)}{非寻常光（e光）}。它们都是线偏振光，且偏振方向相互垂直。
\begin{Itemize}
    \item o光：折射率不变的偏振态，其折射率为$n_o$；
    \item e光：折射率随入射光方向变化的偏振态，其折射率随入射方向在$n_o$和$n_e$之间变化。
\end{Itemize}

其中，$n_o$和$n_e$被称作主折射率。在介质中，光速正比于折射率的倒数，因此对于双折射，o光的光速$v_o$恒定，e光的光速随入射光方向在$v_o$和$v_e$之间变化。
\begin{law}[光轴、o光与e光的偏振方向]
    光轴：光线沿某一方向传播时，o光与e光的折射率、光速等都相同，这便是光轴的方向。光轴是一个方向而非一条线。

    o光的偏振方向垂直于光轴，三维中o光的波面是球；e光的偏振方向在光轴所在平面内且垂直于o光的偏振方向，三维中e光的波面是椭球。
\end{law}
\begin{singlefigure}[正晶体与负晶体]{chapter9_正晶体与负晶体}[0.9]
\end{singlefigure}

把o光和e光的光速随入射方向的变化图画出来如上图所示。左边表示o光的光速大于e光，即o光的折射率小于e光，这类晶体被称作正晶体，典型的代表是水晶、石英等；右边e光的光速大于等于o光的光速，即e光的折射率小于等于o光，这类晶体被称作负晶体，典型的代表有方解石。$z$轴为光轴，沿光轴入射的o光和e光折射率相同，垂直光轴入射的o光和e光折射率之差最大。

在偏振方向上，上图将三维情形取$xOz$作为切面，展示了平面内的o光和e光的电矢量的偏振方向。从偏振分析中也可以看出，垂直于纸面的光偏振态不随入射角改变，因此其折射率和光速也不会被改变；在直面内的光的偏振态会随入射角发生变化，因此其折射率和光速随入射角变化也是合理的。

在针对双折射晶体的分析中，基于惠更斯作图法的分析方法非常直观，可以快速分析出o光和e光的光线方向。惠更斯作图法是基于惠更斯原理的。根据惠更斯原理，波前上每一点都可视为一个子波源。这里假设平行光入射，对于各向同性材料（包括各向异性材料中的o光），具体步骤如下图（假设上方为空气，折射率为1；下方材料折射率为n），以界面与某一条入射光线的交点为起点做光线的垂线，这条线即上方介质中光的波前（相位相等），那么只要据此确定下方介质中光的波前并作垂线，就是折射后的光线。确定波前即确定下方材料中的等相位面，从图中可以看到$A$点与$B_n$点等相位，与$B'_n$点间光程差为$AB'_n\cdot \sin i$，因此在下方材料中与$B'_n$点等相位的$A'$点与$A$点间的光程为$AB'_n\cdot sini$，则$AA'\cdot n=AB'_n\sin i$，因此以$A$为圆心，$\dfrac{AB'_n \sin i}{n}$为半径画圆，由于光波前垂直于光线，因此，过$B'_n$点做圆的切线，得到的就是下方材料中光的等相位面，再过$A$点做该线的垂线，即得到光线的方向。

\[\Delta l=n_e d - n_o d\quad\Rightarrow\quad \Delta \varphi=\dfrac{2\pi}{\lambda}(n_e-n_o)d\]

如果$\Delta \varphi = \pm\dfrac{\pi}{2}$，即$(n_e-n_o)d=\pm\dfrac{\lambda}{4}$，则称该波片为$\dfrac{\lambda}{4}$片，如果入射光为45°偏振，那么经过$\dfrac{\lambda}{4}$片后，其变为圆偏振；$\Delta \varphi = \pm\pi$，即$(n_e-n_o)d=\pm\dfrac{\lambda}{2}$，则称该波片为$\dfrac{\lambda}{2}$片。利用波片进行线偏振与圆偏振转化的示意图如下图。

最后提一下散射中的偏振。散射可以视作原子分子对光的吸收和发射。由于光是横波，即电场振荡方向与光传播方向相垂直，所以当垂直于光的传播方向观察散射光时，就有一个偏振态的光无法辐射，因此观察到的为线偏振光。这一过程的物理解释与布儒斯特角的产生原因类似。所以我们看到的蓝色的天空其实是散射过的不完全偏振光。
